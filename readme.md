# Banking app

## Installation

Clone or download the .zip version of the repository

```bash
git clone https://github.com/kneitukas/coding_task.git
cd coding_task
npm install
```

## Usage

To run the app pass 2 arguments (path to entry file) (path to input file)
```bash
node main.js sample.json
```

To run all tests
```bash
npm run test
```
