# Banking app

## Installation

Clone or download the .zip version of the repository

```bash
git clone https://github.com/kneitukas/coding_task.git
cd coding_task.git
npm install
```

## Usage

To run the app pass 2 arguments [<entryPath>, <inputPath>]
```bash
node main.js sample.json
```

To run all tests
```bash
npm run test
```
